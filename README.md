# Teste FrontEnd Conta Azul

> Teste para vaga de frontend na ContaAzul.

### Requisitos

> node  ^0.10.29

> bower  ^1.3.12

## Setup de desenvolvimento

### Instalar dependências

```shell
npm install
```

### Watch para mudanças
```shell
grunt watch
```

### Gerar distribuição
```shell
grunt dist
```
